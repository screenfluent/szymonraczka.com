# Grilling

Interview the user one branch at a time until the decision tree is resolved.

## Loop

For each unresolved branch:
1. State the decision being resolved.
2. Ask one question.
3. Provide 2–3 realistic options.
4. Recommend one option clearly.
5. Explain why not the others.
6. Wait for the user's answer.

Do not dump all questions at once.
Do not proceed after a vague answer.
Resolve dependencies between decisions in order.

## Output style

Be concise.
Be skeptical.
Surface concrete consequences.
Prefer smaller reversible choices unless the user is making a deliberate foundation decision.
